Zxbase Dart API client.

## Features

- Registration and pairing functionality:
  - Registration.
  - Pairing.
  - Obtaining peers.
  - Obtaining channels.

## Getting started
In _pubspec.yaml_:
```yaml
dependencies:
  zxbase_api_client: ^1.0.0
```

In your code:
```
import 'package:zxbase_api_client/zxbase_api_client.dart';
```
